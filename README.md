# Unbiasor

Cognitive bias awareness tool — helping you make better decisions.

Understand which biases affect your thinking in real-life decision contexts, and learn strategies to counter them.

## Stack

- **Astro** — static site generation
- **Tailwind CSS** — styling
- **Neon** — PostgreSQL (build-time only)
- **Cloudflare Pages** — hosting

## Development

```bash
npm install
npm run dev
```
